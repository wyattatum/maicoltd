# Maico LTD Project for STOCK MANAGEMENT SYSTEM

## Overview
This repository contains both backend and frontend implementations, organized into two main folders:

- **backendOverview/**  
  Contains all server-side logic, including controllers, models, middleware, routes, views, configuration, and supporting files.

- **frontendOverview/**  
  Contains the client-side application, including React components, public assets, and configuration files.

## Backend
The backend is structured in MVC pattern with controllers, models, middleware, and routes. It handles authentication, transactions, and API endpoints.

## Frontend
The frontend is built with React, including components for login, signup, products, and transactions. It uses Vite for bundling and includes configuration files for ESLint and build setup.

## How to Run
- Navigate to `backendOverview/` and install dependencies:
  ```bash
  cd backendOverview
  npm install
  npm start
- Navigate to frontendOverview/ and install dependencies:
    ```bash
    cd frontendOverview
    npm install
    npm run dev
